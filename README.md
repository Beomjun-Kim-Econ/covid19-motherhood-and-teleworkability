# How COVID-19 Lockdown Effect to Mothers? 
This is my final project for an undergraudate course 'Empirical Applied Microeconomics'.  

I was inspired by the work of Claudia Goldin, who is the winner of 2023 Nobel Prize, getting recognized her contribution for identifying the reason of the wage gap between female and male. She points out the nonlinear wage structures, which disproportionately reward long inflexible working hours, are the main source of the wage gap. Mothers continue to bear the majority of childcare responsibilities, making them find 'flexible' jobs, which pay lower.  
  
COVID-19 lockdown might seemingly have relieved this discriminative situation since it made people work remotely, regardless of the gender. Their (presumably non-adult) children also studied home as well. Therefore, I investigate the result of this environment.  
Interpreting Goldin(2014)'s conclusion in light of current Korean context, it would be proper to compare mainly mothers to non-mother female. Because if we compare mothers to all non-mothers, omitted variable concern, which would be accompanied by gender, possibly arises. I conclude this DiD result as the main result.
